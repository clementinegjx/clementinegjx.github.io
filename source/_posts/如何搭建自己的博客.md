### 创建gitee仓库
+ clone下我的仓库吧：
```bash
git clone https://gitee.com/mathieulin/blog.git
```

+ 安装node以及npm
[如何安装node和npm](https://blog.csdn.net/weixin_46030002/article/details/126939600)

+ 进入仓库后安装npm包
```bash
npm install 
```

+ 然后再创一个gitee仓库
创建完后将_config.yml中的deploy下的repo改成你的仓库名
root也要改成对应的
然后在仓库中，点击服务下的Gitee Pages, 把强制使用HTTPS给打开了

### 可以开始了
进入项目中，三部曲进行部署
```bash
hexo clean
hexo g
## 本地运行
hexo s
## 部署
hexo d
```
